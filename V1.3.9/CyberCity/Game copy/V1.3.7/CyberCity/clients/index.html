CYBERCITY BATTLE RULES

Objective: Defenders protect the city while Hackers try to compromise locations.

Victory Conditions: Control more locations than your opponent when the game ends.

Game Flow:

  • 10 rounds total, 2 turns each
  • Defender gets first 2 turns to set up defenses
  • Hacker gets final round advantage
  • Each location worth 1 point (Lackland worth 2 points)

Defender Actions: Firewall, Virus Protection, Intrusion Detection, User Training

Hacker Actions: Phishing, Virus, Malware

Compromise Levels: 75%+ = Compromised, 100% = Permanently lost
